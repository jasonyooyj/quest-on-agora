# Agora - AI 토론 교육 플랫폼

## 회사 및 제품 소개

**Agora**는 대학교와 교육기관을 위한 AI 기반 토론 교육 플랫폼입니다. 고대 그리스의 민주적 공론장 "아고라"에서 영감을 받아, 모든 학생이 AI 튜터와 1:1로 심층 토론하며 비판적 사고력을 키울 수 있는 혁신적인 교육 솔루션을 제공합니다.

---

## 해결하는 문제

### 한국 대학 수업의 현실

한국 대학생들의 수업 참여 실태는 심각합니다:

| 현상 | 실제 모습 |
|------|----------|
| **출석만 하는 수업** | 노트북 켜놓고 유튜브, 쇼핑, SNS... 수업은 배경음 |
| **벼락치기 문화** | 한 학기 내내 안 듣다가 시험 2주 전에 족보로 암기 |
| **수동적 학습** | 교수 말씀 받아적기만, 스스로 생각하는 훈련 부재 |
| **학점 = 암기력** | 비판적 사고, 논증 능력은 평가받지 못함 |

> "수업 듣는 척만 하다가, 시험 때 족보 외워서 학점 받고, 졸업하면 다 까먹는다"

이런 환경에서 학생들은 **'왜?'라고 질문하는 법**, **논리적으로 주장을 펼치는 법**을 배우지 못한 채 사회에 나갑니다.

### AI 시대, 더 심각해진 문제

ChatGPT의 등장으로 상황은 더욱 악화되었습니다:

| 현상 | 실제 모습 |
|------|----------|
| **과제 = 복붙** | ChatGPT에 과제 던지고 결과물 그대로 제출 |
| **생각의 외주화** | "AI가 알아서 해주니까" 스스로 고민하지 않음 |
| **표절 판별 불가** | 교수도 AI가 쓴 건지 구분하기 어려움 |
| **사고력 퇴화** | 검색-복붙에 익숙해져 깊이 생각하는 능력 상실 |

> "요즘 학생들 과제는 다 비슷해요. ChatGPT 문체가 다 똑같거든요."

AI는 도구로 잘 쓰면 강력하지만, **생각을 대신해주는 용도**로 쓰이면서 학생들의 사고력은 오히려 퇴보하고 있습니다.

### 기존 토론 수업의 한계

토론 수업은 이 문제의 해답이 될 수 있지만, 현실적인 제약이 많습니다:

| 문제점 | 현실 |
|--------|------|
| **소수 학생 독점** | 30명 수업에서 실제 발언하는 학생은 5명 미만 |
| **논리적 깊이 부족** | "그냥 그렇게 생각해요"로 끝나는 피상적 토론 |
| **평가의 어려움** | 교수가 모든 학생의 사고 과정을 파악하기 불가능 |
| **참여 장벽** | 내성적인 학생, 외국인 학생의 참여 저조 |
| **딴짓할 틈** | 발언 안 하면 그냥 앉아서 딴짓 가능 |

## Agora의 솔루션

Agora는 **"딴짓할 수 없고, 복붙도 안 되는 구조"**를 만듭니다:

### 왜 복붙이 안 되는가?

| 일반 AI (ChatGPT) | Agora AI 튜터 |
|-------------------|---------------|
| 답을 바로 준다 | **질문만 한다** |
| 결과물을 생성한다 | 학생이 **스스로 생각하게 유도**한다 |
| 복붙 가능 | 실시간 대화라 **복붙 불가능** |
| 누가 썼는지 모름 | 교수가 **전 과정을 모니터링** |

### 핵심 차별점

- **모든 학생이 동시에** AI 튜터와 1:1 심층 대화 → 방관자가 없음
- AI가 **"왜?"**라고 질문하며 논리적 근거 요구 → 대충 넘어갈 수 없음
- 교수는 **실시간 대시보드**로 전체 학생 모니터링 → 누가 참여하는지 다 보임
- **대화 기록이 남음** → 사고의 과정 자체가 평가 대상
- **실시간 진행** → AI에게 대신 시킬 수 없음, 본인이 직접 생각해야 함

> "Agora의 AI는 답을 주지 않습니다. 계속 '왜요?', '그 근거는요?'라고 물어요. 학생이 스스로 생각하지 않으면 대화가 진행이 안 됩니다."

### 결과

| 기존 방식 | Agora |
|----------|-------|
| 시험 = 암기력 테스트 | 토론 과정 = 사고력 평가 |
| 결과물만 평가 | **과정을 평가** |
| 복붙으로 통과 | 본인이 생각해야 통과 |
| 졸업 후 다 까먹음 | **사고하는 습관**이 남음 |

1. **대학교 교수/강사**
   - 철학, 윤리학, 법학, 경영학, 사회학 등 토론 중심 과목
   - 대형 강의에서 학생 참여를 높이고 싶은 교수
   - AI 복붙 과제에 지친 교수

2. **교육기관**
   - 대학교, 대학원, 법학전문대학원
   - 토론/논술 학원, 기업 연수원

3. **기업 교육 담당자**
   - 임직원 비판적 사고 교육
   - 리더십 개발 프로그램

- **철학과**: "인공지능에게 도덕적 지위를 부여해야 하는가?"

## AI 토론 모드

교수가 수업 목적에 맞게 AI의 대화 방식을 선택할 수 있습니다:

### 1. 소크라테스 모드 (Socratic)
> "왜 그렇게 생각하세요? 그 전제가 맞다면..."

- 날카로운 질문으로 숨겨진 가정 탐구
- 철학, 윤리학 수업에 적합

### 2. 균형 모드 (Balanced)
> "찬성 측은 이렇게 주장하고, 반대 측은..."

- 다양한 관점을 균형있게 제시
- 사회 이슈, 정책 토론에 적합

### 3. 토론 모드 (Debate)
> "그런데 반대로 생각해보면..."

- Devil's advocate로 논리적 반론 제기
- 법학, 비즈니스 케이스 스터디에 적합

### 4. 최소 개입 모드 (Minimal)
> "그러니까 ~라고 생각하시는 거군요"

- 학생 발언을 미러링하며 최소 개입
- 자유 토론, 브레인스토밍에 적합

## 사용 흐름

### 강사 입장

```
1. 회원가입 → 강사로 역할 선택
2. "새 토론 만들기" → 주제, AI 모드 설정
3. Join Code 발급 (예: ABC123)
4. 학생에게 코드 공유 (QR코드, 링크)
5. 실시간 대시보드에서 모니터링
6. 토론 종료 → 분석 리포트 확인
```

### 학생 입장

```
1. Join Code 입력 (로그인 불필요)
2. 닉네임 입력 & 입장 선택 (찬성/반대/중립)
3. AI 튜터와 1:1 대화
4. 대화 종료 → 최종 입장 & 근거 제출
5. (선택) 갤러리에서 다른 의견 열람
```

## 경쟁 우위

### vs ChatGPT / 일반 AI
- **답을 주지 않음**: 질문으로 사고 유도, 복붙 불가능
- **교육 목적 설계**: 토론 구조, 입장 선택, 근거 제시 프로세스
- **강사 통제권**: 실시간 모니터링, 개입, 전체 대화 기록 확인
- **과정 평가 가능**: 결과물이 아닌 사고 과정을 볼 수 있음

### vs 기존 LMS
- **실시간 상호작용**: 단순 게시판이 아닌 AI와의 동적 대화
- **개인화**: 각 학생에게 맞춤형 질문과 피드백
- **참여 장벽 해소**: Join Code로 즉시 참여

### vs 오프라인 토론
- **동시 참여**: 100명 이상이 동시에 심층 토론 가능
- **기록 보존**: 모든 대화 내용 저장 및 분석
- **공정한 참여**: 내성적인 학생도 동등하게 참여
